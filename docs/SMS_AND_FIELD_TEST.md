# SMS και πρακτική δοκιμή μαθήματος

## Γρήγορη δοκιμή χωρίς πραγματικό SMS

Το default είναι:

```txt
SMS_PROVIDER=mock
```

Με αυτό δεν στέλνονται χρεώσιμα SMS. Όταν πατήσεις **Αποστολή SMS links σε όλους**, τα μηνύματα εμφανίζονται στο **SMS outbox** της σελίδας μαθήματος και μπορείς να κάνεις copy το link.

## Δοκιμή με κινητά στην ίδια αίθουσα

Ο μαθητής δεν μπορεί να ανοίξει `localhost` του laptop. Βάλε δημόσιο ή LAN URL στο `.env`:

### Επιλογή Α — ίδιο Wi‑Fi

Τρέξε:

```bash
uvicorn app.main:app --host 0.0.0.0 --port 8000
```

Βρες το IP του laptop, π.χ. `192.168.1.45`, και βάλε:

```txt
PUBLIC_BASE_URL=http://192.168.1.45:8000
```

Τα κινητά πρέπει να είναι στο ίδιο Wi‑Fi.

### Επιλογή Β — ngrok / δημόσιο link

Τρέξε app local και μετά άνοιξε tunnel:

```bash
ngrok http 8000
```

Βάλε στο `.env`:

```txt
PUBLIC_BASE_URL=https://το-ngrok-url
```

Έτσι τα links δουλεύουν και από mobile data.

## Πραγματικό SMS με Twilio

Βάλε στο `.env`:

```txt
SMS_PROVIDER=twilio
TWILIO_ACCOUNT_SID=...
TWILIO_AUTH_TOKEN=...
TWILIO_FROM_NUMBER=+...
PUBLIC_BASE_URL=https://το-public-url
```

Με trial Twilio συχνά χρειάζεται να είναι verified οι αριθμοί παραληπτών. Για αύριο, πρώτα κάνε mock/outbox ή ngrok + copy link.

## Ροή ημέρας μαθήματος

1. Πας στον κόμβο.
2. Ανοίγεις το τμήμα.
3. Πατάς `+ Προσθήκη` για νέο μάθημα.
4. Ανοίγει η σελίδα παρουσιών.
5. Πατάς `Αποστολή SMS links σε όλους`.
6. Ο μαθητής ανοίγει το link και βλέπει QR.
7. Πατάς `Άνοιγμα κάμερας / Scanner` και σκανάρεις το QR.
8. Όποιος δεν έχει κινητό καταχωρείται από `Χειροκίνητα` με αιτιολογία.
9. Πατάς `Κλείδωμα & εκτύπωση`.
10. Όσα έμειναν κενά γίνονται `Απών` και μετά μπορείς να δημιουργήσεις `Αναπλήρωση` χωρίς να αλλάξει η παλιά ημερομηνία.
